# InterviewIQ – AI Interview Intelligence Platform

A full-stack AI-powered mock interview platform with:

- Firebase Authentication
- Razorpay Payment Integration
- Credit-Based Monetization System
- ATS Resume Compatibility Scoring
- Real-Time Interview Evaluation
- Optional Camera Mode
- Dark Mode Support

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Firebase Auth
- Razorpay
- Tailwind CSS

## Features
- Secure JWT Authentication
- Protected APIs
- Payment Verification via HMAC
- ATS Score Generation
- Interview Performance Analytics

## Setup Instructions

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
npm run dev
